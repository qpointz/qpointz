# In-progress stories

Stories live here once **`STORY.md` has at least one** work item marked **`[x]`**.

New stories start under [`../planned/`](../planned/) until the first checkbox is checked.

On **story closure**, move the folder to [`../completed/YYYYMMDD-<story-slug>/`](../completed/)
(do not delete). See [`../RULES.md`](../RULES.md) — **Story closure**.
