# Overview

Mill is an experimental data integration platform that offers:

- Unified data querying across multiple formats (CSV, TSV, Parquet, Avro, JDBC) using standard SQL
- RESTful service layer for seamless data access and integration
- Universal connectivity through JDBC drivers and Python client libraries
- AI-powered chat interface for natural language data exploration and insights
- Fine-grained security with role-based access control at both record and table levels
